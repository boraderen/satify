# SATify

SATify is a Next.js app for experimenting with reductions to SAT and 3-SAT.

Supported problems:

- k-Clique
- Independent Set
- Vertex Cover
- SAT
- 3-SAT

## Local development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

## Deployment

GitHub Pages is deployed through the workflow in `.github/workflows/deploy-pages.yml`.
